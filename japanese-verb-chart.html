五段動詞活用形完整指南

含使用場合、敬語／常體例句對照

未然形あ段（〜わ／か／が…）

表示「尚未發生」的狀態，接否定、受身、可能、使役等助動詞。

飲む → 飲ま　書く → 書か　急ぐ → 急が　話す → 話さ
否定（常體）

今日は飲まない。

今天不喝。

否定（敬語）

今日は飲みません。

今天不喝。

※ 敬語否定從連用形「み」接「ません」

否定過去（常體）

昨日は飲まなかった。

昨天沒喝。

否定過去（敬語）

昨日は飲みませんでした。

昨天沒喝。

連用形い段（〜い／き／ぎ…）＋ 音便

用途最廣，接ます、て形、た形，也可名詞化。

飲む → 飲み　書く → 書き　急ぐ → 急ぎ　待つ → 待ち
て形／た形時發生音便：飲んで・書いて・待って
ます形（敬語）

水を飲みます。

喝水。

過去（敬語）

昨日、水を飲みました。

昨天喝了水。

て形接續（常體）

薬を飲んで、寝た。

吃了藥，然後睡覺。

過去（常體）

昨日ビールを飲んだ。

昨天喝了啤酒。

名詞化

飲み物・読み方・休み

飲料・讀法・休假

辭書形う段（原形）

字典形態，常體結句或接各種文法句型。

飲む・書く・急ぐ・話す・待つ・取る・歌う（原形即是）
習慣（常體）

毎朝コーヒーを飲む。

每天早上喝咖啡。

接「とき」

薬を飲むとき、水を使ってください。

吃藥的時候請用水。

接「前に」

寝る前に歯を磨く。

睡前刷牙。

接「つもり」

来年、日本へ行くつもりだ。

我打算明年去日本。

命令形え段（〜え／け／げ…）

語氣強烈，無敬語版本。正式場合改用「〜てください」。

飲む → 飲め　書く → 書け　急ぐ → 急げ　走る → 走れ
緊急指令

早く逃げろ！

快逃！

體育競技

もっと走れ！

跑快一點！

禮貌替代說法

こちらへどうぞ。／～てください。

請往這邊。／請～。

意向形お段＋う（〜おう／こう…）

表達意圖或邀請對方一起做某事。

飲む → 飲もう　行く → 行こう　書く → 書こう　待つ → 待とう
自我意圖（常體）

よし、頑張ろう。

好，來加油吧！

邀請（常體）

一緒に行こう！

一起去吧！

邀請（敬語）

一緒に行きましょう。

我們一起去吧。

接「とする」

出かけようとしたとき、電話が鳴った。

就在正要出門時，電話響了。

假定形え段＋ば（〜えば／けば…）

接「ば」構成條件句，表示「如果～的話」。

飲む → 飲めば　行く → 行けば　急ぐ → 急げば　待つ → 待てば
條件（常體）

毎日練習すれば、上手になる。

如果每天練習的話，就會變厲害。

條件（後句敬語）

早く寝れば、元気になりますよ。

早點睡的話，就會有精神喔。

反事實假設

もっと勉強すれば、合格できた。

當時若更努力念書就能考過了。

慣用「よければ」

よければ、一緒に食べませんか。

如果你不介意的話，要不要一起吃？

受身形あ段＋れる（〜われる／かれる…）

表示「被～」，分直接受身、迷惑受身，也可表尊敬。

飲む → 飲まれる　書く → 書かれる　叱る → 叱られる　呼ぶ → 呼ばれる
直接受身（常體）

財布を盗まれた。

錢包被偷了。

直接受身（敬語）

先生に名前を呼ばれました。

被老師叫了名字。

迷惑受身（常體）

隣の人に騒がれて、眠れなかった。

被旁邊的人吵鬧，睡不著。

尊敬語用法

先生はもう帰られました。

老師已經回去了。（受身形表敬意）

可能形え段＋る（〜える／ける…）

表示「能夠做～」的能力或可能性。

飲む → 飲める　書く → 書ける　急ぐ → 急げる　待つ → 待てる
能力（常體）

漢字が読める。

能讀漢字。

能力（敬語）

日本語が話せます。

會說日文。

不可能（常體）

辛くて食べられない。

太辣了吃不下去。

不可能（敬語）

今日は参加できません。

今天無法參加。

正式替代說法

泳ぐことができる。

能夠游泳。（書面語）

使役形あ段＋せる（〜わせる／かせる…）

表示「讓／叫某人做某事」，有強制與許可兩種語意。

飲む → 飲ませる　書く → 書かせる　急ぐ → 急がせる　待つ → 待たせる
強制（常體）

子供に野菜を食べさせた。

讓（強迫）孩子吃了蔬菜。

強制（敬語）

部下に資料を作成させました。

讓部下製作了資料。

請求許可（敬語）

少し休ませてください。

請讓我稍微休息一下。

使役受身（被迫）

上司に残業させられた。

被上司強迫加班了。

禁止形辭書形＋な

直接禁止某行為，語氣強硬。無敬語版本，正式場合改用「〜ないでください」。

飲む → 飲むな　入る → 入るな　触る → 触るな
強烈禁止

触るな！

不准碰！

親近叮嚀

無理するな。

不要勉強自己。

禮貌替代說法

ここで写真を撮らないでください。

請不要在這裡拍照。

最禮貌說法

駐車はご遠慮ください。

敬請避免停車。
